얼간이들 공용 repo

소개 문구: 
'얼간이들'은 2024년에 만들어진 한국공대 게임공학과 소모임으로서,
게임을 플레이하고, 그것을 넘어 게임을 기획하고, 개발하고자 합니다.

무모하더라도 계속 도전하고, 
실패하더라도 그 실패에서 배우고자 합니다.

2024년부터 게임 기획, 게임 개발 및 스터디, 게임개발 정보 공유 등을 정기적으로 진행하고 있으며,
게임행사 참여, 게임플레이 등의 다양한 활동을 진행하고 있습니다.

2025년 12월부터 게임개발 연합동아리 unidev에 합류해 해당 동아리의 활동을 같이 하고 있습니다.
